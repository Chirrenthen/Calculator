# 🚀Calculator

A sleek and modern calculator web app designed for simplicity, functionality, and style. Perform basic arithmetic operations effortlessly with a user-friendly interface, now also available as a **mobile application**! 📱✨

---

## ✨ Features

- 🧮 **Basic Arithmetic Operations**: Add, subtract, multiply, divide, and calculate percentages.
- 🔘 **Control Buttons**:
  - `AC`: Clear all input at once.
  - `DEL`: Delete the last entered character.
  - `=`: Get instant results with real-time evaluation.
- ⚡ **Real-Time Calculations**: Dynamic and quick updates with every input.
- 💯 **Percentage Support**: Automatically replaces `%` with `/100` for accurate results.
- 🎨 **Interactive Design**:
  - Clean, centered layout for a modern look.
  - Grid-based button arrangement for seamless navigation.
  - Dynamic scaling of buttons for tactile feedback.
- 🛡️ **Error Handling**: Prevents invalid input like starting with operators (`+`, `-`, etc.).
- 🌐 **Lightweight & Optimized**:
  - Minimal CSS for fast loading.
  - Pure JavaScript for real-time functionality without external dependencies.

---

## 🛠️ Technologies Used

- **HTML**: Structured content for the web interface.
- **CSS**: Clean, modern, and responsive design styling.
- **JavaScript**: Core functionality for calculations and dynamic updates.

---

## 📱 Mobile Application

Take the calculator with you anywhere! 🌟 The **mobile app** offers the same features optimized for your phone, delivering fast and intuitive calculations on the go. 📲

1. Clone or download this repository.
